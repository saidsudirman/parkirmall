#  Sistem Informasi Biaya Parkir Mall

Proyek web sederhana untuk menghitung biaya parkir kendaraan di mall.

---

# Fitur
- Pilihan jenis kendaraan: Mobil, Motor, Truk
- Perhitungan durasi otomatis
- Diskon 10% untuk Member
- Penyimpanan data ke data/data.json
- Tampilan menggunakan Bootstrap
